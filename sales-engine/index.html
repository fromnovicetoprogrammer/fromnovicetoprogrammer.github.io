Sales Engine

10 October 2014

Holey moley they were right about this project being extremely intense and definitely the hardest thing yet! Sales Engine, so far, has been a lot of trial and error to see what will work and produce the right information because this is really the first project to not have any interface in which we interact with. So far, its going really well and we are making good progress with the program. While we're done with all of the parsing of the data to something Ruby can use and basic relationships, now is the hard part, because this weekend we will start the Business Intelligence which is so much harder than the parsing. In Business Intelligence we have to take the data from the parser and create a relationship network to pull the data from multiple locations to provide the asked for information. While I know it will not be easy, I am definitely up for this next challenge!
