Dinner Dash

21 November 2014

So this week we started dinner dash the project is going to be the lastest yet, three whole weeks with the same four people. In dinner dash we will be building a restaurant that has a checkout process, cms and secure admin dashboard. We decide on Crossroads Lodge a very cliche mountain lodge serving wild game like bison, bear and mountain lion. This week we mainly concentrated on authorization and authentication, with pushing logic down the stack. My group seems to work really well with each other and have fallen into quite a good workflow. But I hope everything will continue with this efficiency.
