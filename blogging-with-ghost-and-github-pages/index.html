Blogging with Ghost and Github pages

22 October 2014

I decided to under go the feat of creating a blog with my exploits at Turing. I initially decide to do blogger since I already had a Google account, but after a short amount of time I realized that it was not going to allow me to do as much as would like to. So I did some research and found Ghost, a very nice and easy to use blogging service that had all the add-ons and custom themes I could ever want.

I loved the Ghost platform but I did not like the fact that I had to pay $10 a month to run a blog. When people call me a geek I say, “Thank you!” I took away from this and looked for free hosting and blogging platforms. I found that Ghost could be hosted on a server, be it local or web, and went on to looking for a way to host it on github pages. I found two blogs that explained how the process went but neither of them were what I was needing.

In prefacing the instructions, I should tell you that I am running a mac so depending on what rig you're using, it might be slightly different.

Start by downloading the enviroments:

  • Install Node
    Download Node
  • Install Python
    If you have have homebrew run
    brew install python
    or Download Python 2.7.8
  • Use pip to install buster
    pip install buster
  • And lastly, use wget which is required by buster with
    brew install wget

Okay so now that we have everything installed that we need to run the blog lets get the fun part started.

Download the ghost platform to run locally.

  • Create a directory that all blog will live in.
  • Download Ghost
  • Unzip and go to the directory
  • Again in your terminal run to install the server
    npm install
  • And then to run the server locally run
    npm start
  • Visit http://localhost:2368/ to see that everything is working correctly.

So now that we have the blog running locally, we will get it set up on github with buster.

Now to create a Github page for your blog

  • Go to you github profile and create a public repo with the name of your account(for me is the repo name was larsonkonr.github.io) but it you really want to i created a seperate account with the name of what i wanted the blog to be.
  • In the same directory that you put the ghost files setup the location for the static files with
    buster setup
  • and when promted enter the repo you just created for me that is https://github.com/larsonkonr/larsonkonr.github.io.git
  • once that is completed run this to generate the static files
    buster generate --domain=http://127.0.0.1:2368
  • then cd into static
    cd static
  • Now to get it up on github
    git add .
    git commit -m 'Initial commit'
    git push

All we have to do now is wait! Although github says that you'll only have to wait 10 minutes, I should warn you that I had to wait 26 minutes. After everything it was seriously the hardest wait with constantly clicking refresh.

To change any of the styling you have to edit them in the ghost directory. Boot up your local server and add or change any post you would like. To get that on your hosted blog you need to run buster generate --domain=http://127.0.0.1:2368 and then add, commit, and push to github.

There you have it! Your own blog using ghost hosted on github.

Konr Larson
